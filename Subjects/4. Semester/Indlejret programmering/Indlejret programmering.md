> - Viden
> 	- Basic ARM microcontroller architecture

> 	- General purpose input and output, displays, keypads, encoders

> 	- Scheduling, tasks and threads 
> 	- Real-time operating systems for microcontrollers 
> 	- Semaphores, mutex, synchronization and critical sections  
> 	- Deadlocks 
> 	- Fixpoint and floating-point arithmetic 
> 	- Interrupts: reentrancy and recursion 
> 	- Debugging in integrated development environments 
> 	- Assembly in C 
> 	- Compilation and the build process
> - Færdigheder
> 	- Developing software for embedded systems in C 
> 	- Debugging and compiling software for embedded systems 
> 	- Division of problems into tasks and inter-task communication 
> 	- Using peripherals of microcontrollers: keypad, display, encoders, etc.
> - Kompetencer
> 	- Analyze the requirements for an embedded system and develop real-time software 
> 	- Analysis of real-time embedded software 
> 	- Problem-solving 
> 	- Working in teams 
> 	- Communication of results
