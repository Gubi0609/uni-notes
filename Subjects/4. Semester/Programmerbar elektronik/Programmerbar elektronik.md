**Course description:** 
Digitalt design:
- Design af logiske kredsløb ud fra sandhedstabeller, funktionstabeller og booleske ligninger. 
- Standard kombinatorisk logik: Dekoder, Enkoder, Multiplekser.
- Sekventiel logik: SR-latch, D-latch, D-F/F
- Design og dokumentation af synkrone tilstandsmaskiner.

Det hardware beskrivende sprog VHDL:
- Entity og Architechture
- Concurrent og Sekventiel kode
- Kombinatoriske og Clock styrede processer
- Signaler og Variable
- Kodningsformer: Struktur, RTL og Behavorial
- Simuleringsmodeller

Programmerbar logik FPGA:
- Configurable Logic Blocks og Look Up Tables samt den generelle opbygning
- Block RAM og Multipliers
- Ip-cores

Udviklingsværktøjer til digitalt design:
- Vivado
