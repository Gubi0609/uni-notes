For a Finite State Machine Calculator the following components will be needed
- Adder / Subtractor
- Multiplier
- Register (to hold the inputs for later use in calculation or holding the result for later use)
- Some type of FSM
- Shift register (or another way to shift an input)
	- Could be a multiplier instead. E.g. Input 1: **2**, Input 2: **1**. Inputted number should then be **21**. For this, we can multiply 2 by 10 and add 1.
