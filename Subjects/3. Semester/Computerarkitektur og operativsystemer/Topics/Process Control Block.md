**PCB** - **P**rocess **C**ontrol **B**lock

![[Pasted image 20251107091501.png]]

This is the information that the operating system stores about the running processes in order to make the best possible use of the system's resources.
In short, it stores everything running in the program.

## Scheduling queues
![[Pasted image 20251107092000.png]]
When running multiple processes, we can queue the processes and their PCB.

The difference between _ready_ queue and _wait_ queue, is whether the process is ready to be run or not.
